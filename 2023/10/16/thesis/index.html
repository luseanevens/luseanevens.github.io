deal with space target TFR is more likelihood between processing NLP or CV tasks, it shows the perfect periodic frequency modulation in time, and also have relation between each pixel, so we can’t just use transformer model on it because of the output is over the input dimensions, like we have 3 components in the signal, then the output should be [3,input_dimension], and we also can’t use Vison Transformer directly, because of the relationship of the signal is only on the column, which means we can divide the TFR to independent columns then encoder decoder it(much more cost).

  • Ideal 1: use ViT, divide the TFR to some columns, and input to encoder, MLP and decoder.

In mask R-CNN, we can delete the original Retnet part with replace by SSA, because our signal is continuing on the time and SSA with suitable singular value will do denoise and window frequency domain. and also the change is we do not know how many signal component in the signal, so we can’t set a special label, make cross point belong to them both.
