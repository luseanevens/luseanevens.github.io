数据类型

Java是一种强类型语言Strongly typed language，其关键在于变量类型转换。

强类型：指不允许隐式变量类型转换

弱类型：允许隐式变量类型转换

所谓隐式类型转换，就是变量没有明显的类型转换代码，一般弱类型语言会用隐式类型转换：你想把变量当成什么类型来使用，就可以当成什么类型，语言的解析器就会自动的进行转换。

显式类型转换，则是强类型语言使用的，会显示出类型转换的过程。一开始定义一个变量时，就会确定数据类型，如果不进行强制转换，就会保持不变。

Java中主要有八大数据类型，四大类：

  1. integer type 整数类型
  2. floating type浮点型
  3. boolean type 布尔值
  4. character type 字符，字符串为其中的子集String

复习：计算机储存单位数据

  1. bit ：位，计算机中储存的最小单位
  2. byte ： 八个二进制位组成一个字节
  3. word： 在32位的操作系统中，一次性处理数据量称为字，而其处理四个字节，即32位

至于一个字到底有多少字节，其决定于CPU构架，在不同的学科里由于研究的目标不同，字长均不相同。

Integer type

  • Byte : 数据直接保存在一个byte中，其存储范围-128 -> +127

  • Short：保存在两个字节中，范围-32768 -> +32767

  • Int：四个字节，范围-2147483648 -> +2147483647在正负二十亿左右，是最常使用的

  • Long：八个字节 -9223372036854775808-> 9223372036854775808

这里与C语言的区别在于，C语言在不同架构的机器中，其数据类型int或者long的长度，是不同的，也就是说，针对不同平台的程序是不可重用的，而JAVA因为有JVM的缘故，在编译时会将程序翻译成标准的字节码，所以数据类型的意义在任何情况下都能保持不变。

Floating-Point type

  • float：4 byte小数点后6-7位

  • double：8 byte小数点后15位

相当与精确度进行了变更。

The Character type

  • Cher：字符 用单引号表示 ‘A’
  • String： 字符串 用双引号表示“A”

这里注意区别，在Python中一般没什么问题，但是在Java的很多应用场景中，不能混用。

编码标准：

  • GB18030：信息技术中文编码字符集，是中国的国家标准
  • ASCII：基于拉丁字母的编码规则，美国信息交换标准代码America standard Code Information Interchange，其目的是显示英语
  • ISO：其中包含欧洲、中东多个国家各个语系多种语言的多种编码方式
  • Unicode：其中收录了几乎所有的字符集

The Boolean type

布尔类型：True or False

需要区分，在C++中，0同等与False，但是在Java中，0就是数值，不会像C,C++中存在这种转化。
