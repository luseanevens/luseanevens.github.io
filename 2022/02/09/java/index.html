环境配置

不同系统下的基本环境配置

  • macos
  • Linux
  • Windows
  1. 从华为云下载JDK
  2. 配置环境变量：JAVA_HOME、CLASS_PATH、JAVAC_PATH
  3. 从清华大学镜像站下载eclipse

在eclipse安装完成之后就可以直接运行了。

KALI在起初就安装了OpenJDK作为Brupsuite的运行环境，为了多端同步，我仍然希望其能够使用JDK SE 12.0.2 。使用Deb包直接安装会与原本的环境起冲突，因此需要使用tar包解压到指定目录，再替换掉默认的JAVA环境。

华为云中的tar包存在损坏，这点从文件大小就看的出来，因此需要到官网下载。

效果如下所示，可以看到Openjdk已经换为了Java se版本。

之后配置eclipse：先从清华大学开源镜像站下载tar包，选定目录进行解压。

选择Standard VM之后在JRE HOME中选择之前安装JDK的非root目录，这里我将其拷贝到了usr/bin/jdk-12.0.2，之后选择这个环境并接受即可。
