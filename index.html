screen

screen is a command-line utility in Linux that allows you to create and manage multiple virtual terminal sessions within a single terminal window or console. With screen, you can run multiple shell sessions, switch between them, and detach/reattach them without losing the running processes.

36354.ttys003.cheriums-MacBook-Air (Detached) 1 Socket in /var/folders/jq/m584ym1n7ns19bgzlc7x0wjc0000gn/T/.screen

so, the id of screen u just created is: 36354.ttys003.cheriums-MacBook-Air

Then, u can do some job from this session and deteach from it throgh keyboard shortcut:

ConTrol+A+D.

if u want to close it:

other commands:

  • screen: starts a new screen session
  • screen -r: reattaches to a running screen session
  • screen -ls: lists all screen sessions
  • Ctrl+A, c: creates a new screen window
  • Ctrl+A, n: switch to the next screen window
  • Ctrl+A, p: switch to the previous screen window
  • Ctrl+A, d: detaches from a screen session
  • exit: exits the current shell session in the screen window

交叉验证

交叉验证cross-vaildation是一种评估泛化性能的统计学方法，它比单次划分训练集和测试集的方法更加稳定、全面。在交叉验证中，数据被多次划分，并且需要训练多个模型。

最常用的交叉验证方法是k折交叉验证k-fold cross-vaildation，其中 k 是由用户指定的数字，通常取 5 或 10。

5折交叉验证：
1、将数据划分为（大致）相等的 5 部分，每一部分叫作折（fold）
2、训练一系列模型，每折轮流作为测试集评估精度，其他作为训练集训练模型
